# 🎨 paint.py
 
Programa de dibujo construido con Python y la librería turtle. Se pueden dibujar distintas figuras geométricas con la capacidad de usar colores distintos.

## 📋 Cambios por commit
 
### Commit 1 — Original
Primer commit con el código original de paint.py con la paleta de colores básica.
 
### Commit 2 — Nuevo color y círculo
Se agregó el color morado a la paleta de colores y la función circle para dibujar círculos.
 
### Commit 3 — Rectángulo y triángulo
Se añadieron las funciones rectangle y triangle para completar el set de figuras geométricas básicas.
